# Development Documentation

Internal development documents for the CPG-to-ACP project. Not intended for user-facing documentation (see `docs/` for that).

**NOTE:** Everything in this folder should be viewed as a point-in-time reference. Files may not be updated as development moves forward and new decisions are made.

## Structure

| Directory | Contents |
|-----------|----------|
| `design/` | Component and contract design documents |
| `spikes/` | Technical investigations and decision records |
| `ui/` | UI/UX spikes and wireframes (Phase 4) |

Top-level files: project plan, project proposal, and intro deck.
